# l2fprod-common-debs
Build Debian Package for l2fprod-common

**Build dependencies**

- debhelper (>= 9)
- cdbs
- default-jdk
- maven-debian-helper (>= 1.5)
- libjcalendar-java (>= 1.3.2)

**Tested**

- Debian wheezy
- Debian jessie
